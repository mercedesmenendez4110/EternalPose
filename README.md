# EternalPose

## Dependencias de librería Arduino

Este proyecto requiere la librería oficial de Arduino GFX para funcionar con la pantalla Waveshare ESP32-S3 CO5300.

Instala la librería siguiente en el Library Manager de Arduino IDE:

- `GFX Library for Arduino`
https://github.com/moononournation/Arduino_GFX
Si usas Arduino CLI, asegúrate de tenerla en tu sketchbook libraries o en el entorno de Arduino.

No es necesario subir esta librería al repositorio; basta con que quien compile el proyecto la instale localmente.
